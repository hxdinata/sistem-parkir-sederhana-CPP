Deskripsi singkat

    Aplikasi terminal C++ untuk mengelola masuk/keluar kendaraan secara sederhana.

Fitur

    Tambah kendaraan masuk
    Keluarkan kendaraan dan hitung durasi/tarif (jika diimplementasikan)
    Tampilkan daftar kendaraan aktif

Build & Run

    Linux/macOS: g++ Code/*.cpp -o sistem_parkir -std=c++17 ./sistem_parkir
    Windows (MinGW): g++ Code\*.cpp -o sistem_parkir.exe -std=c++17 .\sistem_parkir.exe

Struktur

    Code/ — kode sumber (.cpp/.h)
    README.md — dokumentasi
